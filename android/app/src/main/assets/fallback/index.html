AnyClaw APK 빌드 완료

이 APK는 현재 커밋에서 자동 생성된 Android 패키지입니다. 앱은 먼저 http://127.0.0.1:18923 서버를 시도하고, 서버가 아직 준비되지 않은 경우 이 안내 화면을 표시합니다.

  • android/scripts/build-server-bundle.sh가 프런트엔드/CLI 번들을 APK 자산에 포함합니다.
  • android/scripts/download-bootstrap.sh가 Termux bootstrap zip을 APK 자산으로 내려받습니다.
  • GitHub Actions는 모든 push에서 APK를 빌드하고 Release에 첨부합니다.
